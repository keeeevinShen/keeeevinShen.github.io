Reflection on Building Web App

August 6, 2025
Debugging LangGraph Libraries

Use a Virtual Environment and Keep Libraries Updated

Use a venv and make sure to update the library to the latest version. Or the version you are pretty sure that would work.

The Most Frustrating Two Days

This is probably the most frustrating two days in my life. I was working on this langGraph agent and testing it locally. The weird thing is when I am testing the sending chat function, the afternoon it works but the same day night, it's not working, getting some misleading error message from langGraph.

Almost Giving Up

After struggling two days trying to fix the router and the graph logic itself, and right before the moment I almost decide to quit CS, I upgrade the langGraph library, and.... it works...

失去了所有力气和手段

Sometimes the simplest solutions are the hardest to see. After exhausting every debugging technique, diving deep into complex graph logic, and questioning my entire approach, the answer was just a library version update. It's a humbling reminder that in software development, environment setup and dependency management are just as important as the code itself.
