July 19, 2025

The Power of Using Existing Knowledge

Using the knowledge we know to learn new is one of the most effective ways. Like I've take EECS 398 and now learning SQL, I was struggling in SQL, but once I found its very similar to Pandas I learnt from EECS 398, I all of a sudden got the idea of SQL. The thing is I was spending time understanding this Pandas pattern, now I don't need to do the work again. It would be a lot easier if someone at the beginning just told me this SQL is very similar to Pandas.

The Need to Study and Pass a Course Easily. No Bullshit.

We usually find the courses we took before can actually summarize in a few sentences. So for those courses we just want to pass and get rid of it, why would we spend a lot of time on those unimportant things? The solution would be having someone know all the things and just tell you what you need to know to pass this course.
